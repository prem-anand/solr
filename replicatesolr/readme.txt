SOLR is a commonly used open source enterprise search platform that offers dynamic clustering, rich document handling, real-time indexing, database integration, faceted search, hit highlighting and geospatial search. SOLR monitors search and navigation feature of popularly known interest websites.
This project is about replicating solr for future use. A python script is written to replicate the solr. 


Download 

replicatesolr.py 

Replicatesolr uses the solr replication protocol to get a consistent snapshot. It is not guaranteed that the data that is stored is
consistent. though this can be considered to be an approximate replication.
